# CA2 RISC-V Processor Project

This repository contains the full implementation of the Computer Architecture (2) final project.

The project is developed in multiple phases during the course and includes:

- A custom RISC-V assembler implemented in C++
- A simplified RISC-V processor implemented in Verilog HDL
- Pipeline stages
- Hazard detection and forwarding
- Branch prediction
- Cache simulation
- Performance analysis

## Repository Structure

- `phase1_assembler/` → C++ assembler
- `phase2_verilog/` → processor modules and pipeline stages
- `phase3_enhancements/` → hazards, forwarding, branch prediction, cache
